# 📊 Amazon Sales Dashboard (Power BI)

## 📌 Overview

This project is a Power BI dashboard built using an Amazon sales dataset. It provides insights into sales performance, customer behavior, and business trends.

## 🚀 Features

* Total Sales, Orders, Quantity (KPIs)
* Average Order Value (AOV)
* Sales by Category
* Sales Trend Over Time
* Payment Method Distribution
* Order Status Analysis
* Top Products by Sales
* Sales by State (Map)
* Interactive Slicers (Category, Payment, Date)

## 🛠 Tools Used

* Power BI
* DAX (Data Analysis Expressions)
